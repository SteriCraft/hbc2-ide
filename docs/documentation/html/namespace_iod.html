Byte  peripheralId,
unsigned int  nbPortsRequested 
)

Requests HbcIod for available ports.

HbcIod keeps track of the peripheral id plugged in the ports.
This function is called on initialization of peripherals.
It triggers 2 interrupts for the CPU to identify plugged devices.

Returns an empty vector of sockets if there is not enough available ports

Parameters
nbPortsRequestedNumber of ports requested
Returns
a vector of the sockets created (empty if there is not enough available ports)
