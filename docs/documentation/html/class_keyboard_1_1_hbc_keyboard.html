Detailed Description

Derived from HbcPeripheral, represents the Keyboard device.

This class is responsible for emulating the HBC-2 keyboard.

Device ID: 0xAA

Communication

Like every HbcPeripheral, HbcKeyboard uses sockets connecting it to HbcIod ports to send its scan code to the HbcCpu.

List of available ports
ID Port Description
0 PRESSED_SCAN_CODE Holds the scan code of the pressed key
1 RELEASED_SCAN_CODE Holds the scan code of the pressed key
