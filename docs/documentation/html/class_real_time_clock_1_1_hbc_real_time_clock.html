Detailed Description

Derived from HbcPeripheral, represents the RTC device.

The Real Time Clock (or RTC) is a self-powered device that stores and keeps track of date and time. See QDate and QTime for more information about date and time storage.

List of available ports
ID Port Description
0 YEAR Year value (byte representing [1900 - 2155])
1 MONTH Month value
2 DAY Day value
3 HOUR Hour value
4 MINUTE Minute value
5 SECOND Second value
6 CMD Command sent by the CPU
List of available commands
ID Command Description
0 NOP No operation
1 GET_TIME Sends current date and time on ports
1 SET_TIME Sets internal date and time with data available on ports

Any invalid command will result in NOP.
