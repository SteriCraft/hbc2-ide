Detailed Description

Singleton of the configuration manager, that handles IDE settings.

Reads and writes settings in the configuration file.
The configuration file is stored in the system's default app config directory.

For example, on Linux this file can be found here: /home/<username>/.config/HBC-2_IDE/config.cfg
